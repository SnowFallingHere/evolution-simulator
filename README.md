# 进化模拟器 - 完整指南

## 网址 ：https://snowfallinghere.github.io/evolution-simulator/

## 🎮 游戏介绍

《进化模拟器》是一款基于Web的生存进化模拟游戏，玩家将引导一个生物从原始状态开始，通过捕猎、探索、思考等活动不断进化，最终达到智慧的巅峰。游戏融合了生存策略、资源管理和角色成长元素。

## 🌟 核心特性

- **多阶段进化系统**：从原始生物到智慧生命的完整进化历程
- **丰富的状态管理**：饥饿、疾病、心理健康等多维度状态
- **冷却机制**：所有活动都有合理的冷却时间
- **主题切换**：支持明暗两种主题模式
- **开发者控制台**：隐藏的调试工具（需要特殊方式解锁）

## 🎯 游戏目标

将你的生物从进化等级0培养到100级，解锁所有能力，建立社交关系，制作工具，最终达到智慧的顶峰。

## 🕹️ 基本操作

### 主要活动按钮

| 按钮 | 冷却时间 | 解锁条件 | 效果 |
|------|----------|----------|------|
| 获取进化点数 | 无冷却 | 始终可用 | 点击获得进化点数 |
| 进行捕猎 | 10秒 | 始终可用 | 获得食物，可能受伤 |
| 进行休息 | 5秒 | 始终可用 | 恢复健康，减少疾病 |
| 进行蛰伏 | 8秒 | 始终可用 | 大幅恢复，属性可能下降 |
| 探索远处 | 7秒 | 等级≥6 | 发现食物或进化点数 |
| 进行锻炼 | 8秒 | 力量≥20 | 提升力量和速度 |
| 进行思考 | 12秒 | 智慧≥45 | 提升智慧，解锁高级功能 |
| 与其他物种互动 | 15秒 | 等级≥51 + 已思考 | 合作/竞争/杀戮获得不同收益 |
| 尝试制作工具 | 20秒 | 等级≥51 + 已思考 | 成功提升智慧，失败减少心理健康 |
| 尝试交友 | 10秒 | 等级≥51 + 已思考 | 成功提升心理健康，可能获得食物 |

### 状态监控

**属性状态：**
- 力量值：影响捕猎效果和工具制作
- 速度值：影响捕猎效果
- 智慧值：影响思考和高级活动成功率

**健康状态：**
- 饥饿值：达到100会导致死亡
- 心理健康值：51级后思考解锁，达到0会导致死亡
- 疾病值：达到100会导致死亡

**资源状态：**
- 食物储存：自动消耗，影响饥饿值增长
- 进化点数：用于升级进化等级

## 🔄 进化阶段

### 初级阶段 (1-50级)
- 只有基本生存活动
- 隐藏智慧和心理健康
- 死亡条件：仅饥饿和疾病

### 探索阶段 (6级解锁)
- 解锁"探索远处"功能
- 开始接触更广阔的世界

### 智慧阶段 (51级解锁)
- 显示智慧属性（初始1.0）
- 可以感受到心理健康状态
- 解锁思考和锻炼功能

### 社会阶段 (思考后解锁)
- 显示心理健康值（初始50）
- 解锁高级社交功能：互动、制作工具、交友
- 死亡条件增加：心理健康值≤0

## ⚙️ 开发者控制台

### 解锁方式
在**10秒内连续点击主题切换按钮（🌙/☀️）10次**即可解锁控制台。

### 控制台功能
- **页面切换**：快速切换到不同游戏页面
- **时间控制**：暂停/恢复游戏时间
- **属性修改**：直接设置力量、速度、智慧等属性
- **状态控制**：调整饥饿、疾病、心理健康值
- **进化控制**：设置进化点数和等级
- **需求查看**：查看所有等级的需求点数

## 🎮 游戏策略

### 初期策略 (1-5级)
1. 优先点击"获取进化点数"积累进化能量
2. 合理使用"进行捕猎"维持食物供应
3. 适时"进行休息"恢复状态
4. 食物不足时使用"进行蛰伏"减少消耗

### 中期策略 (6-50级)
1. 积极"探索远处"获得额外资源
2. 通过捕猎和探索提升基础属性
3. 注意保持食物储备，避免饥饿
4. 及时休息减少疾病值

### 后期策略 (51级+)
1. 尽快"进行思考"解锁高级功能
2. 使用"进行锻炼"提升战斗属性
3. 通过"与其他物种互动"建立关系
4. "尝试制作工具"提升智慧
5. "尝试交友"维持心理健康

## ⚠️ 注意事项

1. **冷却管理**：合理安排活动顺序，避免同时冷却
2. **状态平衡**：注意饥饿、疾病、心理健康的平衡
3. **资源规划**：食物消耗会随等级增加，提前储备
4. **风险控制**：捕猎和探索有受伤风险，做好准备

## 🛠️ 技术特性

- **响应式设计**：完美支持桌面和移动设备
- **实时状态更新**：所有状态实时显示和计算
- **冷却进度条**：直观的活动冷却显示
- **事件记录**：完整记录所有重要活动
- **自动保存**：状态持续保存，刷新不丢失进度

## 🎉 进阶技巧

1. **保底机制**：当饥饿值≥85时，捕猎和探索成功率大幅提升
2. **属性协同**：力量影响捕猎和工具制作，智慧影响思考和社交
3. **活动连锁**：休息后可以立即蛰伏，形成恢复链
4. **风险收益**：高风险活动（如杀戮）带来高收益但影响心理健康

## 🔧 故障排除

如果游戏出现异常：
1. 刷新页面重新开始
2. 检查浏览器控制台是否有错误信息
3. 确保所有JavaScript文件正确加载
4. 使用控制台功能重置游戏状态
